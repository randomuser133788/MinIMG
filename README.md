MinIMG is an image-viewer program. It has been under development ever since MiniShell's release and has been prepared for public release as of January 1st, 2026.

MinIMG has one bug: the image leaves trails when the MinIMG window is resized.

You may be warned by Windows Defender that MinIMG is untrustworthy on Windows 8 or newer; just run anyway.

MinIMG has been tested on Windows 7, Windows Vista and Windows Preinstallation Environment 3.0 and it works. It hasn't been tested on Windows XP yet.
